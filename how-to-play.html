How to Play Wanzofc Quiz

Getting Started

First, you need an account. If you don't have one, go to the Register page to create one. If you already have an account, simply Login.

The Lobby

Once you are logged in, you will be in the lobby. Here you can:

  • Create a Room: Click the "Create Room" button. You can give your room a name, and select a quiz category and difficulty.
  • Join a Public Room: See a list of rooms created by other players. Click the "Join" button next to any room to jump in.
  • Join by ID: If you have a specific Room ID from a friend, type it into the input box and click "Join Room".
  • View Your Profile: Click the user icon next to your name to see your stats.

Gameplay

Once in a room, the game will begin shortly. Read the questions and select the answer you think is correct. Be quick, but be careful! The game ends after a set number of questions.
